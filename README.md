# SeR.HSRP

## Avvio rapido

Per esporre la documentazione in locale:

1. Installare i requisiti (solo Python standard).
2. Avviare un web server statico dalla root del progetto:

	```bash
	python3 -m http.server 8000
	```

3. Aprire il browser su `http://127.0.0.1:8000/`.

La homepage (`index.html`) consente di visualizzare l'anteprima e scaricare il file HTML completo via link diretto.
